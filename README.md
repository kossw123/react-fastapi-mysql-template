1. 개념
   “이 시스템은 무엇을 관리하는가?” - 코드가 아닌 현실 세계의 개념을 설명한다.

->


- Kisok system은 사용자를 등록하여 상품을 등록하고 선택하여 결제하는 시스템이다.
    - 상품은 재화를 통해 서비스나 물건을 등록한 사용자에 한해 구매할 수 있다.
    - 사용자는 아이디와 비밀번호를 가지며 로그인을 통해 kisok 페이지에 접근할 수 있다.
    - kiosk 페이지는 상품을 진열하는 페이지이며 결제할 상품을 선택할 수 있다.
    - 상품등록은 kiosk 페이지의 우상단에서 진행하고, 이름과 가격을 기입하여 상춤을 등록한다.
